Spotify Randomizer

John Conway's Game of Life

To gain experience with RESTful APIs, I chose to create a web application using Spotify's and Genius's resources. With Flask Login the user can create an account and add their favorite Spotify artist's to their own designated database. The database was created with postgreSQL and is stored on Heroku. The Spotfiy API allows users to have access to artist's album artwork, snippets of music, song titles, and much more. This web application utilizes the Flask framework, along with Python in its' back-end. To create the user interface, I soley used HTML, CSS, and Jinja. Jinja allows for Python like expressions while ensuring that templates are evaluated in a sandbox. The backend also uses the Genius API to allow the user to view artist lyrics based on the current song playing from the website.

Spotify Randomizer Website

My program allows a user to create a user name and log in. The Web application then allows for a user to save specific artist IDs from spotify to listen to a shuffled mix of various artists that they add to their specific database. To use my website, create a username, log in, and then paste your favorite artist's artist Spotify URIs within the search box. The website will then allow you to listen to the artists you've added along with allowing you to view their Genius lyrics, album art, and song names.
